A list of most expensive <a href="https://www.poewiki.net/wiki/Transfigured_skill_gem">transfigured gems</a> (0 quality, level 1, non-corrupted) in the game <a href="https://www.pathofexile.com/">Path of Exile</a>

Made by fetching `poe.ninja/api`, filtering for transfigured gems and sorting by price.

Updated every 6 hours

## Usage

Go to https://maciejszaman.github.io/poegems/
